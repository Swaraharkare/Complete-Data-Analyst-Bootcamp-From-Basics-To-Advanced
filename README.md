## 📁 Repository Architecture & Engineering Modules

### 🐍 1. Python Foundations & Data Analysis (`/01_Python_Analysis`)
* **Python Basics & Libraries:** Foundations of syntax, control structures, and array operations using Pandas and NumPy.
* **Data Analysis with Python:** Practical data cleaning, structural merging, and dataset transformation workflows.

### 🧪 2. Statistics for Data Science (`/02_Statistics`)
* **Descriptive Statistics:** Core metrics for tracking data distribution, central tendency, shapes, and variance.
* **Probability Distributions:** Mathematical data modeling through Probability Distribution Functions (PDFs).
* **Inferential Stats & Hypothesis Testing:** Rigorous data validation utilizing confidence intervals and A/B testing frameworks.

### ⚙️ 3. Feature Engineering & Exploratory Data Analysis (`/03_Feature_Eng_EDA`)
* **Feature Engineering:** Preprocessing pipelines for missing value mitigation, categorical encoding, and feature scaling.
* **Exploratory Data Analysis (EDA):** Applied visual diagnostics using Matplotlib and Seaborn to isolate outliers and underlying data patterns.

* **SQL Server Foundations:** Core relational querying, normalization, table structures, and troubleshooting questions.
* **SQL Functions:** Structural transformations using internal system, scalar, and aggregate operations.
* **Advanced SQL & Interview Prep:** High-performance queries using CTEs, Window functions, and real-world interview business cases.

### 📊 5. Power BI & Microsoft Excel Foundations (`/05_PowerBI_Excel`)
* **Power BI Core Environment:** Visual design principles, Power Query ETL configurations, and custom DAX data modeling.
* **Core Power BI Dashboards:**
  * *Sales Data Analysis:* Relational star schema design tracking corporate sales performance.

  * *UPI Transactions Data Analysis:* Tracking high-volume fintech payment trends and network failure points.
* **Microsoft Excel Analytics:** Designing business dashboards using Power Query, pivot models, and automated live data imports from SQL Server.

### 🗺️ 6. Tableau Analytics Ecosystem (`/06_Tableau_Analytics`)

* **Tableau Production Dashboards:** Interactive spatial and analytical reporting layers (Dashboards 1 & 2).
